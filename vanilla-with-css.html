GET 200 - Success

Request returned a 200 OK status indicating success.

POST request returned 201 Created, indicating resource creation.

GET 400 - Bad Request

Request could not be understood by server due to malformed syntax.

GET 401 - Unauthorized

Authentication is required, returned 401 Unauthorized.

GET 403 - Forbidden

Valid request but action is not allowed, returned 403 Forbidden.

GET 404 - Not Found

Requested resource does not exist, server returned 404.

GET 500 - Internal Server Error

Server encountered an error and returned 500 status.
